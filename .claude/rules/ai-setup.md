# AI setup maintenance

When editing any file in scope, prefer `/beutl-ai-self-review` over hand edits if the change is part of a broader review. The skill batches related updates and keeps decisions consistent across `AGENTS.md`, agents, skills, rules, and docs.

If you have just finished a substantial task — 3+ files changed, a new feature, a non-trivial refactor, or a PR review cycle — and have not invoked `/beutl-ai-self-review` yet, do that before declaring the task done.
